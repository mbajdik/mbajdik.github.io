Definíció: Számegyenes egy részhalmaza

  • Zárt intervallum: tartalmazza a határpontjait is (Jelölés: [A;B], szögletes zárójel)
  • Nyílt intervallum: nem tartalmazza a határpontjait (Jelölés: ]A;B[ vagy (A;B), a fordított szögletes zárójel vagy általános zárójel)

Műveletek intervallummal:

  • Összeg (Unió):
    • [1;3[ ∪ ]2;6] = [1;6]
    • ]-3;1] ∪ {-3} = [-3;1]
  • Különbség:
    • ]0;5] \ [2;6[ = ]0;2[
    • ]-2;7] \ [2;4[ = ]-2;2[ ∪ [4;7]
    • ]-1;4] \ {3} = ]-1;3[ ∪ ]3;4]
    • [5; 9] \ {9} = [5;9[
  • Metszet:
    • [2;5[ ∩ [3;6] = [3;5[
    • [-7;-4] ∩ {-5} = {-5}
