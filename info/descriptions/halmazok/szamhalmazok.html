Definíció: a számtartomány számokból álló halmaz, röviden számhalmaz; egy szám tulajdonságai alapján sorolható be egy számhalmazba

Nevezetes számhalmazok jelölései:

  • ℝ → Valós számok halmaza
  • ℚ → Racionális számok halmaza (két szám hányadosaként felírható szám)
  • ℚ* → Irracionális számok halmaza (ℚ komplementere, ha U = ℝ)
  • ℤ → Egész számok halmaza (egész számok)
  • ℕ → Természetes számok halmaza (0 és ℤ+ vagy ℤ+0)

Nevezetes számhalmazok további jelölései:

  • Felső index + → Pozitív számok
  • Felső index — → Negatív számok
  • Alsó index 0 → Nullát is tartalmazza
