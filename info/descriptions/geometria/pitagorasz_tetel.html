Pitagorasz tétel: Derékszögű háromszög befogói oldalhosszának négyzetösszege egyenlő az átfogó hosszának négyzetével

a2 + b2 = c2

Bizonyítás (lásd a képén):

  • 1. Négyzet: egy “a” és egy “b” oldalú négyzetből áll és négy darab egybevágó derékszögű háromszögből; T = 4Tháromszög + a2 + b2
  • 2. Négyzet: négy darab egybevágó derékszögű háromszögből és egy “c” oldalú négyzetből áll; T = 4Tháromszög + c2
  • Végső képlet:
    • T1 = T2
    • 4Tháromszög + a2 + b2 = 4Tháromszög + c2
    • egyszerűsítve a2 + b2 = c2

Pitagorasz tétel megfordítása: Ha egy háromszög két oldalhosszának négyzetösszege egyenlő a harmadik oldal hosszának négyzetével, akkor a háromszög derékszögű
