Olyan függvény amely “x”-hez “x” abszolútértékét rendeli (képe V-alakú)

Jellemzés (f(x) = |x|):

  • Értelmezési tartomány: ℝ
  • Értékkészlet: [0; ∞[
  • Zérushely: 0
  • Tengelypont: 0
  • Szélsőérték:
    • Minimum: hely: 0; érték: 0
    • Maximum: nincs
  • Monotonitás:
    • ]–∞; 0] szigorúan monoton csökken
    • [0; ∞[ szigorúan monoton nő
  • Paritás: páros
  • Konvex függvény
