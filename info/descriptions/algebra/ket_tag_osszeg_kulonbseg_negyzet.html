Tétel: Két tag összegének négyzete egyenlő a két tag négyzetével és a tagok szorzatának kétszeresével (a + b)2 = a2 + 2ab + b2, és két tag különbségének négyzete egyenlő a két tag négyzetével és a tagok szorzata kétszeresének különbségével (a – b)2 = a2 – 2ab + b2

  • Bizonyítás (összeg): (a + b)2 = a2 + ab + ba + b2 = a2 + 2ab + b2

  • Bizonyítás (különbség): (a – b)2 = a2 – ab – ba + b2 = a2 – 2ab + b2 (szorzás tényezői felcserélhetők)
