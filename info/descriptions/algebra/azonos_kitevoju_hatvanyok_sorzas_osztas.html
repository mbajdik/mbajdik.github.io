Szorzása: Azonos kitevőjű hatványok szorzásakor a két szám szorzatát hatványozzuk (an * bn = (a * b)n)

  • Bizonyítás: a3 * b3 = a * a * a * b * b * b = (a * b) * (a * b) * (a * b) = (a * b)3 (szorzatok esetén zárójellel szabadon lehet "csoportosítani")

Osztása: Azonos kitevőjű hatványok osztásakor a két szám hányadosát hatványozzuk (an / bn = (a / b)n) (a tört hatványozásának visszaalakítása)

  • Bizonyítás: a2 / b2 = (a * a) / (b * b) = (a / b) * (a / b) = (a / b)2 (újracsoporosítjuk a műveleteket, a tört hatványozásának visszafordítása; feltétel, hogy a ≠ 0)
